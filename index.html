Coma Bem

Sistema para gerenciar informações e comercialização de frutas e verduras

Resumo TCC

O objetivo é apresentar desde a concepção do projeto até sua implementação e lançamento no mercado, abordando escolha de tecnologias, processo de desnevolvimento e estratégias de marketing. O sistema busca oferecer um serviço diferenciado, valorizando a saúde e bem-estar dos consumidores, promovendo produtos selecionados entregues com agilidade e segurança. O trabalho visa contribuir para o comércio de alimentos saudáveis, proporcionando uma opção confiável para consumidores em busca de praticidade e qualidade.

Objetivo Geral:

Um sistema de vendas online de frutas, verdura e legumes

    Objetivo Específico:

  • O sistema deve cadastrar usuários;
  • O sistema deve permitir o login de usuários;
  • O sistema deve mostrar os produtos;
  • O sistema deve buscar, adicionar, visualizar e excluir produtos;
  • O sistema deve mostrar, finalizar, e cancelar o pedido.


  • O sistema deve atender o cliente de forma rápida e eficiente;
  • O sistema deve carregar rapidamente para evitar que os usuários eperem muito tempo para acessar produtos e informações;
  • O sistema deve estar disponível na parte do determinado para que os clientes possam fazer compras a qualquer momento;
  • O sistema deve ser fácil de navegar, para ajudar os usuários a encontrar produtos com facilidade;
  • O layout do sistema deve se adaptar a diferentes dispositivos, como smartphones e tablets para proporcionar uma experiência agradável em todos os tamanhos de tela.
